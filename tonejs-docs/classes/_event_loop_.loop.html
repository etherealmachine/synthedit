Class Loop<Options>

Loop creates a looped callback at the specified interval. The callback can be started, stopped and scheduled along the Transport's timeline.

example

import { Loop, Transport } from "tone"; const loop = new Loop((time) => { // triggered every eighth note. console.log(time); }, "8n").start(0); Transport.start();
