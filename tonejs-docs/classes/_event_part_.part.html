Class Part<ValueType>

Part is a collection ToneEvents which can be started/stopped and looped as a single unit.

example

import { Part, Synth } from "tone"; const synth = new Synth().toDestination(); const part = new Part(((time, note) => { // the notes given as the second element in the array // will be passed in as the second argument synth.triggerAttackRelease(note, "8n", time); }), [[0, "C2"], ["0:2", "C3"], ["0:3:2", "G2"]]);

example

import { Part, Synth } from "tone"; const synth = new Synth().toDestination(); // use an array of objects as long as the object has a "time" attribute const part = new Part(((time, value) => { // the value is an object which contains both the note and the velocity synth.triggerAttackRelease(value.note, "8n", time, value.velocity); }), [{ time: 0, note: "C3", velocity: 0.9 }, { time: "0:2", note: "C4", velocity: 0.5 } ]).start(0);

    • Defined in tone/Tone/event/Part.ts:183

    Get/Set an Event's value at the given time. If a value is passed in and no event exists at the given time, one will be created with that value. If two events are at the same time, the first one will be returned.

    example

    Parameters
