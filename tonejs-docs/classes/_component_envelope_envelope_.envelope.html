Class Envelope

Envelope is an ADSR envelope generator. Envelope outputs a signal which can be connected to an AudioParam or Tone.Signal.

          /\
         /  \
        /    \
       /      \
      /        \___________
     /                     \
    /                       \
   /                         \
  /                           \
example

import { Envelope, Gain } from "tone"; // an amplitude envelope const gainNode = new Gain(); const env = new Envelope({ attack: 0.1, decay: 0.2, sustain: 1, release: 0.8, }); env.connect(gainNode.gain);

  • Overrides ToneAudioNode.constructor

    • Defined in tone/Tone/component/envelope/Envelope.ts:108

    Parameters

    • Optional attack: Time

      The amount of time it takes for the envelope to go from 0 to it's maximum value.

    • Optional decay: Time

      The period of time after the attack that it takes for the envelope to fall to the sustain value. Value must be greater than 0.

    • Optional sustain: NormalRange

      The percent of the maximum value that the envelope rests at until the release is triggered.

    • Optional release: Time

      The amount of time after the release is triggered it takes to reach 0. Value must be greater than 0.

    Returns Envelope

channelCountMode

  • get channelCountMode(): ChannelCountMode

  • Inherited from ToneAudioNode.channelCountMode

    • Defined in tone/Tone/core/context/ToneAudioNode.ts:158

    channelCountMode determines how channels will be counted when up-mixing and down-mixing connections to any inputs to the node. The default value is "max". This attribute has no effect for nodes with no inputs.

    • "max" - computedNumberOfChannels is the maximum of the number of channels of all connections to an input. In this mode channelCount is ignored.
    • "clamped-max" - computedNumberOfChannels is determined as for "max" and then clamped to a maximum value of the given channelCount.
    • "explicit" - computedNumberOfChannels is the exact value as specified by the channelCount.

    Returns ChannelCountMode

    import { AmplitudeEnvelope, Oscillator } from "tone"; const env = new AmplitudeEnvelope().toDestination(); const osc = new Oscillator().connect(env).start(); env.triggerAttack(); // trigger the release half a second after the attack env.triggerRelease("+0.5");
